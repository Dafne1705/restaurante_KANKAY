# restaurante_KANKAY 
El restaurante KANKAY, ubicado en Cieneguilla, realiza el control de insumos de cocina de manera manual mediante cuadernos físicos. Cada área de cocina (parrilla, frituras y salteados) registra diariamente los productos utilizados, las cantidades retiradas y las cantidades consumidas o vendidas durante la jornada. 

Este proceso depende de la responsabilidad de cada cocinero, quien debe anotar manualmente la información para mantener el control del stock. Debido al crecimiento de las operaciones y la necesidad de contar con información precisa, se ha identificado la necesidad de implementar una solución informática que permita gestionar estos registros de forma más eficiente.
